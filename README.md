# eslint-config-dcard

ESLint config for Dcard projects

## Installation

``` bash
$ npm install eslint-config-dcard --save-dev

# You also have to install eslint-plugin-mocha
$ npm install eslint-plugin-mocha --save-dev
```

## Usage

In `.eslintrc`

``` js
{
  "extends": "dcard"
}
```
